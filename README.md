# 🚗 Road Accident Analysis Dashboard

> An interactive **Power BI dashboard** developed to analyze road accident trends, casualty statistics, vehicle involvement, road conditions, and geographical accident distribution. The dashboard provides actionable insights that help transportation authorities identify high-risk areas and improve road safety.

---

# 📌 Project Overview

Road accidents have a significant impact on public safety and infrastructure. This project transforms raw accident data into an interactive dashboard that enables stakeholders to monitor accident trends, analyze casualty patterns, and identify major contributing factors.

Using **Power BI**, the dashboard provides visual insights into accident severity, road conditions, weather impacts, vehicle types, and location-based accident distribution.

---

# 🚀 Features

- Accident Overview Dashboard
- Casualty Analysis
- Accident Severity Analysis
- Monthly Trend Analysis
- Vehicle Type Analysis
- Road Type Analysis
- Urban vs Rural Comparison

## Monthly Accident Trend

Analyze and compare casualty trends across different months and years to identify seasonal accident patterns and monitor yearly performance.

---

## Vehicle Type Analysis

Casualties are analyzed based on vehicle categories such as:

- Car
- Bike
- Bus
- Van
- Agricultural Vehicle
- Other Vehicles

This helps identify which vehicle types are most frequently involved in accidents.

---

## Road Type Analysis

Accident casualties are compared across different road types including:

- Single Carriageway
- Dual Carriageway
- Roundabout
- One Way Street
- Slip Road

This helps identify road infrastructures associated with higher accident rates.

---

## Urban vs Rural Analysis

Compare accident casualties occurring in:

- Urban Areas
- Rural Areas

The visualization highlights how accident distribution differs across geographical regions.

---

## Light Condition Analysis

Analyze accidents occurring during:

- Daylight
- Darkness

This helps evaluate the impact of visibility on road safety.

---

## Geographic Accident Distribution

An interactive UK map visualizes accident locations, allowing users to identify accident hotspots and regional patterns for targeted safety improvements.
